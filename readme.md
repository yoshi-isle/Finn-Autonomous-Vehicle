Meet Finn, an autonomous mini robotic with the purpose of navigating through complex terrains!

Finn was assembled using the Freenove 4WD Car Kit (Compatible with Arduino IDE) https://www.amazon.com/dp/B07YBQ73CH?ref=ppx_yo2ov_dt_b_fed_asin_title

A simulated 3D world in Unity is included to demo the car's thought process and calculations
